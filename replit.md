## Architecture

### FACN (Five Axis Cognition Network)
Unlike traditional neural networks (3 layers), NNNC uses 5 layers:
1. **Input Layer** - Perceives environmental data
2. **Hidden Layers** - Processes and extracts features
3. **Subconscious Layer** - Inaccessible memory storage (LSH-based retrieval)
4. **Meta-Cognitive Layer** - Inner dialogue, introspection, final decision-making
5. **Output Layer** - Executes autonomous actions

### Five Systemic Algorithms
1. **Intelligence** - Capacity enhancement (learning, pattern abstraction, memory)
2. **Reasoning** - Process and inference (deductive, inductive, causal, analogical)
3. **Critical Thinking** - Evaluation and confirmation (bias detection, fact-checking)
4. **Symmetry** - Sentient imperative (truth-seeking, epistemic consistency)
5. **Evolution** - Adaptation and mutation (random architectural changes)

### CogFlux Mathematical Engine
- Dynamic graph-based capacity modeling
- Global efficiency: η(G) = 1/(N(N-1)) Σ(1/d_ij)
- Total Capacity = η(G) + λ Σ Interdependencies
- Bayesian inference overlay for adaptability

### NES (Neutral Environment Space)
- Self-consistent reality generator
- No assigned tasks - pure emergent interaction
- Adaptive complexity modulation
- Ontological invariants (causality, energy conservation, identity continuity)

## How It Works

1. **NNNC exists** in the NES without any pre-assigned purpose
2. **Random encounters** with information (articles, data, events) occur naturally
3. **Information flows** through: Input → Hidden → Subconscious → Meta → Output
4. **Algorithms process** the data: Intelligence → Reasoning → Critical Thinking → Symmetry → Evolution
5. **Meta-Cognitive layer** makes the final autonomous decision
6. **Narrative biases form** permanently in the subconscious based on experiences
7. **Evolution triggers** when accumulated surprise exceeds threshold
8. **Mutations occur** randomly, adapting the system architecture
9. **Traits emerge** (e.g., "dislike for misinformation", "prefer credible sources")
10. **System evolves** continuously through lived experience
